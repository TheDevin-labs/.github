## 🚀 What is Devin’ Labs?

**Devin’ Labs** is a development ecosystem focused on creating tools, languages, and automation systems.

We build with one core idea:

> Code should be powerful, flexible, and owned by those who create it.

---

## ⚙️ Core Project: Doscript

**Doscript** is our automation-focused scripting language.

* ⚡ Fast and flexible scripting
* 🧩 Designed for automation and tooling
* 🔒 Proprietary core for stability and consistency
* 🛠 Open ecosystem for tools, plugins, and extensions

You are free to:

* Use Doscript
* Build with it
* Distribute and sell what you create
